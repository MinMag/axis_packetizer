== Design Description
This is a AXI-Stream pass-through design that receives incoming parallel data, packages it with a header, sequence count and length indicator, and appends a CRC32 checksum, then sends it out an AXI Stream master interface.

The output Packet will begin with the header, then the sequence count, then the length indicator, then the payload, and then a CRC32 checksum.

== Packet Format
Each packet shall consist of:

* A 2-byte preamble
* A 2-byte packet id
* A 4-byte payload length indicator
* The payload
* A 4-byte CRC

== Control Unit State Machine
The control unit consists of 4 main states, with varied number of substates:

. IDLE
. WRITE_HEADER
. STREAM_PAYLOAD
. APPEND_CRC

== Performance Targets 
Latency of 4 clk cycles

Clk Frequency of 250 MHz
